Description globale

Au cours de mes années de BTS j'ai du effectuer différentes missions que ce soit en atelier professionnalisation ou en stage.

Atelier de professionnalisation

Intitulé de la situation professionnelle : Application Fredi


Objectif : Réaliser un site de gestion de bordereaux


Contexte : Pour aider la M2L nous avions un site web à faire pour permettre aux adhérents et à leurs tuteurs légaux de saisir des bordereaux et de les modifier. Il a ainsi été décidé d'utiliser le Modèle Vue Controleur pour composer l'architecture du site. J'ai ainsi dû m'occuper de la partie mot de passe oublié ainsi que de la page de profil.


Période : Septembre - Décembre 2022


Lien vers les ressources documentaires : Dossier de projet

Intitulé de la situation professionnelle : Application Sauteuhz


Objectif : Réaliser une application de gestion pharmaceutique multi-plateforme


Contexte : La pharmacie SAUTEUHZ souhaite un intranet pour gérer les clients ayant des traitements réguliers qui nécessitent un suivi particulier. Pour se faire nous avons dû créer une API afin de permettre une adaptabilité de l'application quelques soient les plateforme ainsi qu'un framework CSS (Bulma) et un framework JavaScript (React).


Période : Février - Avril 2023
