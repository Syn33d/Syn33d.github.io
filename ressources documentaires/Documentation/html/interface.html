Tout savoir sur l'interface

  • Rechercher des biens immobiliers

    L'interface contient dans la barre de menu une option de recherche nommée "Rechercher" permettant ainsi d'effectuer une recherche ciblée sur un bien immobilier (voir image ci-dessous).

  • Création d'une nouvelle affaire

    L'interface contient plusieurs listes déroulante permettant la création d'une nouvelle vente ou d'une nouvelle location ainsi que la sélection du type de bien immobilier permettant ainsi l'envoie d'un formulaire à remplir.

  • Lancement de l'impression

    L'interface contient dans la barre de menu une option d'impression nommée "Imprimer" permettant d'imprimer le récapitulatif d'une création d'une nouvelle affaire.

  • Exportation d'informations

    L'interface contient dans la barre de menu une option d'Exportation nommée "Exporter" permettant d'exporter le récapitulatif d'une création d'une nouvelle affaire dans un autre fichier au format .csv (tableur).
